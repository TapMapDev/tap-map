# TapMap

## Особенности

- версии Флаттера и Дарта
  - Flutter version 3.24.5
  - Dart version 3.5.4
  - Xcode version 16.2

## Инструкция по установке

### Install dependencies:

Вложить .env в корневую директорию проекта;

   ```sh
   flutter pub get
   ```

### Run application:

-Подключаем Андроид эмулятор или симулятор;
-Запускаем `flutter run`;
